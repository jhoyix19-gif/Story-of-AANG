# 📚 NovelVerse

NovelVerse is a WebNovel-style reading platform built with HTML, CSS, and JavaScript.

## Features

- 🔑 User Login & Registration
- 📖 Novel Reading System
- 📚 Novel Library
- 🏆 Rankings Page
- ✍️ Create Novel Page
- 💬 Comments Section
- ❤️ Like System
- 📱 Responsive Design
- ✨ Netflix-Inspired UI
- 🎨 Anime/WebNovel Theme

## Pages

- index.html
- login.html
- register.html
- dashboard.html
- library.html
- rankings.html
- story.html
- create.html

## Technologies

- HTML5
- CSS3
- JavaScript
- LocalStorage

## Installation

1. Download the project.
2. Open the project folder in VS Code.
3. Run `index.html`.

Or deploy with GitHub Pages.

## GitHub Pages

1. Upload all files to GitHub.
2. Go to Settings → Pages.
3. Select:
   - Branch: main
   - Folder: /root
4. Save.

Your website will be available at:

https://yourusername.github.io/repository-name/
